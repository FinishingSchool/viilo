# viilo

## Install

### 1. Get mongodb

#### OS X

`brew install mongodb`

Make your database directory, just incase you're not cool enough. #JSONLyfe Do you have read and write permissions? No. Well there ya' go sport.

`mkdir -p /data/db`

### 2. pull down node modules

`npm install`

## Running server

Start mongo

`mongod`

In another terminal

`npm start`

## Running tests

Start mongo

`mongod`

In another terminal

`npm test`
